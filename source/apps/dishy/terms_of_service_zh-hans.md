# 食记用户服务条款

**最后更新日期：2025 年 12 月 6 日**

## 1. 条款协议

欢迎使用食记（以下简称"本应用"）。下载、安装或使用食记即表示您同意受本服务条款（以下简称"条款"）的约束。如果您不同意这些条款，请勿使用本应用。

## 2. 用户账号和数据

### 2.1 无需用户账号
食记**不需要**您创建用户账号。本应用完全在您的设备和个人 iCloud 账户上运行。

### 2.2 数据存储
- 您的所有数据都**本地存储在您的设备上**
- 如果您启用 iCloud 同步，数据将存储在**您的个人 iCloud 账户中**（受 Apple 条款约束）
- 我们不维护用户数据库，不在我们的服务器上收集或存储您的信息

## 3. AI 菜谱生成

### 3.1 第三方 AI 服务
当您使用 AI 菜谱生成功能时，我们会将您的食物图片发送给第三方 AI 服务提供商：
- **OpenAI**（GPT 模型）
- **阿里云通义千问**（Qwen 模型）
- **字节跳动豆包**（Doubao 模型）

### 3.2 API 密钥
- 您可以使用应用的默认 API 密钥或提供您自己的密钥
- 您的 API 密钥安全地存储在设备钥匙串中，我们不会访问或存储

### 3.3 免责声明
- AI 生成的菜谱是建议，应根据您自己的判断使用
- 我们不保证 AI 生成菜谱的准确性、安全性或质量

## 4. 高级订阅

### 4.1 付款和自动续订
- 订阅通过 Apple 的 App 内购买系统处理
- 确认购买时，费用将从您的 Apple ID 账户中扣除
- 除非在当前订阅期结束前至少 24 小时关闭自动续订，否则订阅将自动续订
- 您可以在 Apple ID 账户设置中管理订阅并关闭自动续订

### 4.2 免费试用
- 新用户可能有资格获得免费试用期
- 如果您在试用期结束前不取消，您将被收取订阅费用
- 您可以在试用期间随时取消，不会被收费

### 4.3 取消和退款
- 您可以随时通过 Apple ID 账户设置取消订阅
- 取消在当前计费周期结束时生效
- 不提供部分订阅期的退款

- 您保留在应用中创建的菜谱、照片和内容的所有权利
